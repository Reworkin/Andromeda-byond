paintable_decal_category
/datum/paintable_decal_category

Категория окрашиваемых декалей

Хранит набор информации, которую использует покрасчик декалей, чтобы определить, что он может красить, что нет, и как это делать.

Vars

cached_category_dataКэширует данные интерфейса, чтобы избежать их регенерации каждый раз. Они всё равно не меняются
categoryЧитаемое имя категории
default_alphaАльфа-канал для декалей, окрашенных из этой категории (при условии, что не используется пользовательский цвет)
dir_listВарианты направлений для этой категории - в формате читаемая метка - значение dir для возврата
paintable_decal_typeТип окрашиваемой декали, которую содержит эта категория - должен быть подтипом /datum/paintable_decal
possible_colorsВарианты цветов для этой категории - в формате читаемая метка - значение цвета
preview_floor_iconИконка, используемая для предварительного просмотра
preview_floor_stateСостояние иконки, используемое для предварительного просмотра

Procs

generate_all_spritesheet_iconsВозвращает список иконок предварительного просмотра для каждой разновидности каждой декали в этой категории для использования в спрайтшите
generate_colored_decal_spritesheet_iconФактически генерирует иконку предварительного просмотра для конкретного состояния, направления и цвета декали
generate_independent_decal_spritesheet_iconsВозвращает список иконок предварительного просмотра для конкретного состояния и направления декали
get_decal_infoИспользуется покрасчиком декалей для изменения состояния декали в зависимости от... состояния.
get_ui_dataСоздаёт и возвращает данные интерфейса этой категории
is_color_validПроверяет, является ли переданный цвет одним из цветов этой категории
is_dir_validПроверяет, является ли переданное направление одним из направлений этой категории
is_state_validПроверяет, является ли переданное состояние иконки одной из декалей этой категории
spritesheet_keyВозвращает ключ для иконки спрайтшита, используется для избежания дубликатов
